initiation au canvas 2D

je pars du principe que vous savez déjà manipuler les types primaires (Number, Strings, Arrays, Objects)
connaissez les bases de la programmation Javascript (boucles, conditions)
avez déjà utilisé un serveur local, un éditeur avancé ou les devtools.
être familier avec une API de dessin aide un peu
mais sinon, pas besoin de grand chose d'autre :)

  • le plus simple est de cloner ou télécharger le repo quelque part
    créer un serveur local qui pointe vers ce dossier
  • avec un éditeur de code pas trop con ou Node, créez plutôt une tâche planifiée
  • sinon, le REPL du pauvre:
    si vous n'avez pas d'éditeur avancé (sublime, webstorm, eclipse... )
    rendez vous sur l'index du serveur local (cette page)
    ouvrez la console de debug et affichez les sources
    clic-droit sur la fenêtre de navigation des sources(panneau de dorite)
    ajoutez le dossier racine à l'espace de travail
    vous avez à présent un droit en écriture sur les fichiers du serveur local
  • ensuite, choississez une page et selon les cas:
    • lisez les commentaires (pour la première page surtout)
    • décommentez les blocs commentés (/* >//* )
    • commentez les return; qui cassent la boucle de dessin draw()
    ctrl + S ( sauver le doc )
    F5 (recharger la page)
  1. survol des concepts
    • hello world
    • hello world 2 : la bas e qu'on va réutiliser tout le temps
  2. les couleurs (article de MDN)
  3. les primitives:
    • point (rectangle)
    • ligne
    • polygone
  4. les cercles
    • cercles
    • trigo de base ( cos et sin )
    • trigo angle et distance
  5. représenter une variable
    • une variable qui s'incrémente
    • une variabel en 2 dimensions et plus
  6. 3 fonctions vitales pour la dataviz
    • lerp: interpolation linéaire
    • norm: normalisation

exercices

créer des fonctions paramétriques permettant de:
  • couvrir l'écran de points
  • dessiner un carré avec des ronds
  • dessiner un rond avec des carrés
  • animer un point le long d'un cercle.
  • dessiner une grille avec un nombre de colonnes / rangées variable
  • dessiner une grille de largeur / hauteur variable
  • dessiner une grille avec un nombre de colonnes / rangées variable et de largeur / hauteur variable
